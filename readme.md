How to run the program:

English:

- Install Visual Studio 2019.
- Clone/Download the repository.
- Open MyGameWin.sln (or Mac/Linux etc... I only tested Windows though).
- Click Run/Start!
- The game will take a while to be built.
- After the first run, you can go to bin -> Debug -> MyGame.exe to launch the game again.
Make a shortcut if you want lol!

Tiếng Việt:

- Cài đặt Visual Studio 2019.
- Clone/tải code từ GitHub về máy.
- Mở file MyGameWin.sln (hoặc Mac/Linux/... game này chỉ mới được test trên Windows).
- Bấm nút Run/Start (chạy).
- Game sẽ mất một lúc để build file lần đầu.
- Sau lần chạy đầu tiên, bạn có thể tới folder bin -> Debug -> MyGame.exe để chạy game thay vì lặp lại các bước trên.
